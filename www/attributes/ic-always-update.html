ic-always-update - The Always Update Attribute

Summary

The ic-always-update attribute forces an element to always update the DOM when new content is downloaded, even if the current content is identical to the downloaded content.

This can be useful if you want to clean non-DOM related state (such as checkboxes) after an operation causes an update.

Syntax

The value of this attribute can be either true or false

This attribute may be placed on parent elements, allowing you to specify behavior across multiple elements.

Dependencies

No effect.
